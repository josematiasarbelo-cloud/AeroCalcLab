Formulas

Option 1 — Crosswind approach

\[ U_{app} = \sqrt{ U_{rider}^2 + U_{cross}^2 } \] \[ \beta = \tan^{-1} \left( \frac{U_{cross}} {U_{rider}} \right) \]

Option 2 — Wind magnitude + direction

\[ U_{\parallel} = U_{wind}\cos(\theta) \] \[ U_{\perp} = U_{wind}\sin(\theta) \] \[ U_{rel,\parallel} = U_{rider} + U_{\parallel} \] \[ U_{app} = \sqrt{ U_{rel,\parallel}^2 + U_{\perp}^2 } \] \[ \beta = \tan^{-1} \left( \frac{U_{\perp}} {U_{rel,\parallel}} \right) \]
