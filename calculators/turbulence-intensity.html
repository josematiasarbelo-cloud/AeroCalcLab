Formulas

Directional turbulence intensity

\[ TI_x = \frac{u'_x}{U} \] \[ TI_y = \frac{u'_y}{U} \] \[ TI_z = \frac{u'_z}{U} \]

Compound turbulence intensity

\[ TI = \frac{ \sqrt{ \frac{ u_x'^2 + u_y'^2 + u_z'^2 }{3} } } {U} \]
